# PSPpred-GIN
A GIN-based deep learning framework to classify phage structural proteins.

Given a phage protein sequence as input, PSPpred-GIN classifies it one of the 10 following classes:
1. Major capsid
2. Minor capsid
3. Baseplate
4. Major tail
5. Minor tail
6. Portal
7. Tail fiber
8. Tail sheath
9. Collar
10. Head-Tail joining (HTJ)
